## Getting started

1. Clone the repository:
```bash
git clone https://github.com/anastasia-kravchuk/todo-app-react-typescript.git
cd todo-app-react-typescript
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
Run the project locally:
```bash
npm start
# or
yarn start
```
